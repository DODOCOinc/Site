## Snippet WordPress: Ficha D&D 5e

Arquivo: `dnd5e-character-sheet-snippet.php`

**Uso rápido**
1. Cole o conteúdo do arquivo no plugin **Code Snippets** (ou no `functions.php` do tema).
2. Em uma página/post, insira o shortcode: `[dnd5e_sheet]`.
3. A ficha permite salvar no navegador, baixar JSON e salvar no servidor (via `admin-ajax.php`).
